CÁC PHÍM TẮT:
        Ctrl + S : Lưu Link vào [danh sách]
        Ctrl + D : Paste Link vào TextBox và tự động (Ctrl + S)
        Ctrl + BackSpase : Delete Link trong [Danh Sách]
        Ctrl + Spase : Mở File Excel chứa dữ liệu
        Ctrl + M : Bật/Tắt hướng dẫn
        F5 : Chạy
        F6 : Ẩn/hiện giao diện
        Ctrl + Break : Thoát
